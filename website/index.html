MyAI-Tool

Local AI Code Assistant powered by Ollama.
Bluetooth-enabled hardware control.

Get Started Try Bluetooth Demo

Features

Local AI REPL

Chat with local LLM (llama3.2) with tools: file read/write, bash, search.

Code Assistant

Python/Rust fork of CLAW. Edit files, run commands, debug locally.

Bluetooth Control

Web Bluetooth API: Scan, connect, control BLE devices directly from browser.

Easy Install

Ollama + myai.py. No cloud, privacy-first, offline capable.

Bluetooth Demo (Chrome/Edge, enable localhost:8000)

Demo uses Web Bluetooth API. Pair a BLE device and control services.

Quickstart

1. Install Ollama

ollama pull llama3.2:3b
ollama serve

2. Run MyAI

python myai.py

Chat: myai> Hello!

3. Rust CLI

cd rust && cargo run --bin claw-cli
